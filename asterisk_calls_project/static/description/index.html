Asterisk Calls Project

Connect calls to projects and tasks!

Watch the DEMO video

Features

  • Partner's number and dial buttons for click-to-call added to project and task forms.
  • Automatically open project or task form on incoming call.
  • Create project or task from call history record.
  • Access linked project and task during a call from any place.
  • Helpdesk based on projects and tasks can be organized.
  • Custom development available....

Active Calls

Incoming calls are searched agains open tasks and projects for a match.

If a match is found the call is linked to the found task and project.

User can click on a button near the task or project column to open the corresponding form.

Call popup widget makes linked project and task accessible from any place with just a click!
