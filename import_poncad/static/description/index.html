PON CAD is a 3D Scaffolding Design Software BIM developed to help Designers, Renters and Producers of small and big structures.

Import Order and Products from PONCAD csv file with one click

If multi is selected, for each section create a different sale order

Each sale order has the same origin: the cad file

Products are created automatically if not exist the internal reference

PON CAD 3D in a minute

PON CAD improves your work by offering a complete solution to:

  • Bill of Materials and Precise quotations for commercial offers
  • Technical Drawings, assembling schemes and sections for safety at work
  • Structural Analysis of scaffolds, stages and coverings - EN 12811-1, NTC2018 and Eurocode 3 - 9
  • Stock management at the warehouse and the building site
  • Presentations: Videos, Rendering and 3D VR

PON CAD has no limits in designing because it works on standard 3D CAD

  • Small and large buildings
  • Industrial maintenance
  • Support structures
  • Stages and Entertainment
